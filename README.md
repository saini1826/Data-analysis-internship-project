# Data-analysis-internship-project
This repository contains an end-to-end data analysis project using python. The project includes data cleaning, exploratory data analysis (EDA), data visualization, and insights generation. Tools used : Python, Pandas, NumPy, Matplotlib, Seaborn.
## Objective
The objective of this project is to analyze an e-commerce datasets to gain insight into customer purchasing behavior, sales performance, and product trends.
The project focuses on data cleaning, exploratory data analysis (EDA), ana data visualization using Python.
## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
## Project Steps
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Insights Generation
## Conclusion
This project helps understand sales patterns and customer behavior, which can support better business decision-making.
