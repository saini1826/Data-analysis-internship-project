## E-Commerce Data Analysis Project
This repository contains an end-to-end data analysis project using python. The project includes data cleaning, exploratory data analysis (EDA), data visualization, and insights generation. Tools used : Python, Pandas, NumPy, Matplotlib, Seaborn.
## Objective
Analyze e-commerce sales data to understand sales trends, top products, and customer purchasing behavior.
The project focuses on data cleaning, exploratory data analysis (EDA), ana data visualization using Python.
## Analysis Performed
- Data loading and understanding
- Data cleaning and preprocessing
- Feature engineering (Total Sales)
- Exploratory Data Analysis (EDA)
- Data visualization with business insights
## Key Insights
- The UK is the higest revenue-generating country
- A small number of products contribute most of the sales
- Sales show seasonal patterns
## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

- Exploratory Data Analysis (EDA)

## Conclusion
This project helps understand sales patterns and customer behavior, which can support better business decision-making.
This project demonstrates an end-to-end data analysis workflow with a focus on generating actionable business insights.
